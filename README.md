  <h1>나는 무엇일까요</h1>
  안터생태공원 습지생물을 주제로 하여 만든 AR 교육용 앱 <b>나는 무엇일까요?</b>입니다.

  AR Image Tracking을 이용하여 만들었습니다.  
  
  도감창과 포토카드에 쓰인 이미지의 출처는 unsplash와 안터생태공원 교육용 자료입니다.  

  어플 '나는 무엇일까요?'의 플레이 영상은 제목을 클릭하시면 보실 수 있습니다.
